# Action Feedback: <short title of the action>

> Record what actually happened after you acted on a plan or an AI suggestion. This closes the loop and makes the OS smarter over time. Name it like `2026-06-07-published-homepage.md`.

| Field | Value |
|-------|-------|
| **Date** | YYYY-MM-DD |
| **Project** | <project name> |
| **Action source** | Human / AI suggestion / Both |
| **Outcome** | Success / Partial / Failed |

## The action

What did we actually do?

> Example: "Published the homepage using the chosen static site generator."

## Expected result

What did we think would happen?

## Actual result

What really happened? Include concrete details and any surprises.

## What worked

The parts worth repeating.

## What didn't

The parts to avoid or change next time.

## Lesson learned

The reusable takeaway in one or two sentences.

> Promote durable lessons into [`02_Knowledge/`](../starter-kit/10_AI_OS/02_Knowledge/README.md).

## Next action

What this outcome implies we should do next. Feed this back into the Project Map.

## Related

- Links to the Project Map, relevant Decision Records, or Incident Records.
